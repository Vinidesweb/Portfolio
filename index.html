Procurando um Desenvolvedor FullStack?

Vinícius Brust O trabalho é uma das maiores responsabilidades da vida. Trate-o com empenho e dedicação! Vamos revolucionar o mercado de tecnologia juntos?

Marinheiro MR-RC da Marinha do Brasil, sempre engajado na transformação e evolução profissional. Sou iniciante no mercado tecnológico com conhecimento em HTML, com CSS, Javascipt, Typescript, Node JS, React, Git e GitHub, Maria DB, SQL que obtenho através do curso online de Desenvolvedor FullStack no Profissão Programador. Neste curso coloquei em prática o conhecimento que obtive, desenvolvendo um site de compras de ingresso para filmes, no momento, estou praticando o desenvolvimento da rede social YouTube e assim cada vez mais estou tendo experiência em desenvolver mais apps como WhatsApp, Facebook, entre outros. Também estou iniciando um curso online através da parceria do Google + CIEE que é sobre Análise de Dados. Tenho interesse em obter uma graduação no ensino superior de Ciências da computação com intuito de aprender novas habilidades e somar conhecimentos e, assim, agregar valores no mercado de trabalho.

Este foi o primeiro projeto que fiz criando um web site de compras de ingressos do filme Batman. Como foi o primeiro projeto e estava no começo de curso fiz a estrutura e estilização só usando HTML e CSS. Conforme fui avançando e obtendo mais conhecimentos, fui implementando esses conhecimentos melhorando o web site como transformar para o modo responsivo, e depois, melhorei o site utilizando o Javascipt e React.

Este projeto que estou desenvolvendo é a rede social YouTube. No momento estou fazendo a parte de criar um estado global para o menu lateral. Neste projeto estou usanso React, Typescript e também vou criar um banco de dados para atuar na parte de Backend.
